## 1) Contexto do projeto

- Requisito principal: 3 estudos de caso para cada tópico:

- Restrição-chave da setorização: aproximar da metragem do terreno de projeto (**1085 m²**).

## 5) Critérios de aplicação no lote de 1085 m²
- Priorizar zoneamento em 5 setores:
  1. Pedagógico
  2. Apoio terapêutico/sensorial
  3. Administrativo
  4. Serviços
  5. Área externa controlada
- Recomendações TEA + Down:
  - Redução de ruído/reverberação
  - Iluminação natural controlada (sem ofuscamento)
  - Circulação intuitiva com sinalização clara
  - Espaços de descompressão/acolhimento
  - Acessibilidade universal (NBR 9050)
